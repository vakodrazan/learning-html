Tips

Have you ever written a code? If you haven't this website is best for you then. This built for free to everyone who wants to build a website. Learning is good because it helps people to be creative.

The tips you should follow

There are some tips that I would like to share with you.

  • First of all, the file name has to be lowercase. It must be with .html.
  • you need to write a clean code. What is a clean code? Clean code is a good looking and easy to read.
  • You need to inspect you indenting. A child elements need to have spaces. It depends on the indenting sometimes it has two space but sometimes it has four spaces.
  • You should use editorconfig
  • You should always use w3 validation to avoid error. When check your work there choose valid by direct input. Always make sure before you publish it.
  • You need to test your work in chrome to check if it works.

Do not use:

  • Format documents to fix your indenting. It always gives you a bad indenting.
  • Use style. It is good to write a simple code but clean.
  • Mouse but use keyboard shortcut instead.

    What keyboard shortcut should you use?

    Those are the most important keyboard shortcut in html. It is always good to use them when you write a code.

    • CTRL + END is used when you want to jump one word.
    • CTRL + FN + END is used to jump to the end of the documents.
    • When you want to edit a same text at the same time you need to select all then CTRL + D untill you get all of the text you want to edit.
    • When you want to fix the indenting using TAB is faster.
    • If you don't want the writing in one row use toggle word wrap. By clicking CTRL + SHIFT + P then choose toggle word wrap, it will automatically fix your writing in a new line when it is too long.
