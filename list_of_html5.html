List of all HTML5

These are the list of html5 that we need to know if you want to code. These are the most useful.

Sectioning elements

This are all the sectioning elements. We can nest other elements inside but we can't nest it by itself. These are the most popular sectioning elements:

  • <article>
  • <aside>
  • <footer>
  • <header>
  • <main>
  • <nav>
  • <section>

Head elements

These are the list of the head elements that we need to know. All of them need closing tag, except <meta> we don't use closing tag.

  • <meta>
  • <title>
  • <link>
  • <script>
  • <style>

Inline elements

Inline elements can be used in any cases. We can nest it with one another. These are the lists:

  • <a>
  • abbr
  • <address>
  • <b>
  • <cite>
  • <code>
  • <del>
  • <dfn>
  • <em>
  • <i>
  • <ins>
  • <mark>
  • <q>
  • <s>
  • <kbd>
  • <samp>
  • <span>
  • <small>
  • <strong>
  • <sub>
  • <sup>
  • <time>

Form elements

These are the list of form elements I should know. We use them in form.

  • <button>
  • <datalist>
  • <fieldset>
  • <form>
  • <input>
  • <label>
  • <optgroup>
  • <option>
  • <select>
  • <textarea>

Block elements

These are the block elements. We can't put them inside of each other because they are the same level.

  • <h1>-<h6>
  • <p>
  • <ol>
  • <ul>
  • <pre>
  • <dl>
  • <hr>

Table elements

These are the elements you need to know when want to make a table
  • <table>
  • <thead>
  • <tbody>
  • <tfoot>
  • <tr>
  • <th>
  • <td>
  • <caption>
  • <col>
  • <colgroup>
By knowing those elements list, you are good at code.
