Nesting rules in HTML

In HTML, there are some of the elements that are best to nest in other elements. We need to know about the rules when we nest element. However, we need to know the use of the elements.

We need to know

  • We cannot put the same elements inside each other. For example, we can't put <header> inside of <header> elements. However, we can use <header> inside of <main> elements.

  • Do not forget the end tag:

    We should always close the child elements tag before close the parents tag. It can cause error if the end tag of the child tag doesn't appear before the end tag of the parents elements. However, we don't need end tag with void elements. For example: <input>, <meta>

Don't do these:

  • Nesting elements that are the same value such as <ul> inside of <ol>.
  • Using h1-h6 not in order. For example, you use h1 then jump into h3.

Do these instead:

  • When you want to nest <ol> inside of <ul>, we need to enter <li> after <ul> element. Then we can enter the ordered list, because it is allowed as a child of lists.
  • You should use h1-h6 in order. For example, after using <h1>, we should use <h2>.
