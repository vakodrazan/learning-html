Nesting rules in HTML

In HTML, there are elements which are best to nest in other elements. We need to know about the rules when we nest elements. However, we need to know the use of the elements.

We need to know

  • We cannot put the same elements inside each other. For example, we can't put <header> inside of <header> elements. However, we can use <header> inside of <main> elements.

  • Do not forget the end tag:

    We should always close the child elements tag before close the parents tag. It can cause error if the end tag of the child tag doesn't appear before the end tag of the parent elements. However, we don't need end tag with void elements. For example: <input>, <meta>

Don't do these:

  • Nesting elements that are the same value such as <ul> inside of <ol>.
  • Using h1-h6 not in order. For example, you use h1 then jump into h3.

Do these instead:

  • When you want to nest <ol> inside of <ul>, we need to enter <li> after <ul> element. However, we can't do that unless we use the list element first <li>.
  • You should use h1-h6 in order. For example, after using <h1>, we should use <h2>.

Examples:

This is what we should do when we nest <ol> and <h3> inside of <ul>.

<ul>
<li>
<h2>Fruit</h2>
<ol>
<li>Apples</li>
<li>Bananas</li>
<li>Coconut</li>
</ol>
</li>
</ul>
