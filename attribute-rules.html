Attibutes rules

There are lots of attributes that we can use in html. Remember, when use for attribute then you have to use id attribute, and the value have to be the same in both attributes. Also, we need to respect the space between the attributes. For example, you write a name attribute the move one space then write the value attribute.

  • Don't do these:

    Write an <input> element without name attribute.

    It is known that value attribute is an optional, but if you a better code it is good to have a value

    Using uppercase with attributes.

  • Do these instead:

    When you write an <input>, we have to put a name attribute.

    Use lowercase with value attributes.

  • Examples

    Let's take an example on how to use attributes in an input element.

    <label for="fruits">
    <input id="fruits" name="fruits" value="apples" required>
